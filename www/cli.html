Remote Clipboard

Command Line Interface

Use remote-clipboard directly from your terminal by installing the CLI tool:

cargo install remote-clipboard

More details can be found here

The CLI is ideal for environment that don't have a graphical environment, since clipboard are handled by GUI. This tool aims at removing this gap, while keeping the API simple and somewhat secure. Please keep in mind that your content is not identified, meaning anyone can potentially access it or erase it. Unless you purchase and register for a private namespace from the homepage.
